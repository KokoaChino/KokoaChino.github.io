ref和reactive的区别

  • ref：用于创建一个响应式的单一值，适合基本数据类型；ref返回一个对象，其中的值通过.value访问
  • reactive：用于创建一个响应式对象，适合复杂数据结构；reactive 直接返回一个可响应的对象

插入 HTML

将 HTML 字符串插入到页面中，允许动态渲染和显示 HTML 内容

{{ }}会将数据解释为普通文本，而非 HTML 代码

为了输出真正的 HTML，需要使用v-html指令

条件渲染

根据特定条件控制某些元素在 DOM 中的显示或隐藏

v-if有更高的切换开销，而v-show有更高的初始渲染开销

因此，如果需要非常频繁地切换，则使用v-show较好；如果在运行时条件很少改变，则使用v-if较好

列表渲染

根据数组或对象动态生成一组 DOM 元素，方便展示重复的数据项

动态绑定

将 Vue 实例中的数据与 DOM 元素的属性或内容关联起来，实现数据和视图的自动同步

:是v-bind:的简写形式

双向绑定

实现数据与视图之间的自动同步

修饰符

.lazy

在默认情况下，v-model在每次input事件触发后将输入框的值与数据进行同步

可以添加lazy修饰符，从而转为在change事件之后进行同步

.trim

自动过滤用户输入的首尾空白字符

事件监听

响应用户的交互，通过绑定事件处理函数来执行特定的操作，增强用户体验和应用的互动性

事件命名同DOM事件

@是v-on:的简写形式

侦听器

监控 Vue 实例中的数据变化，并在数据发生变化时执行特定的回调函数

组件

组件是 Vue 应用的基本构建块，封装了特定功能和界面的独立、可重用的代码片段

每个组件可以拥有自己的数据、模板、样式和生命周期方法，便于组织和管理复杂的用户界面

组件化开发提高了代码的可维护性和复用性

一个组件就是一个.vue文件（子组件）

加载组件

  1. 引入组件：import MyComponentVue from './components/MyComponent.vue'

  2. 挂载组件：components: { MyComponentVue }

  3. 显示组件：<my-componentVue/>

组件生命周期

  1. **created**：【创建】组件实例被创建后调用，此时数据观测已设置，但 DOM 还未挂载
  2. **mounted**：【渲染】组件挂载到 DOM 后调用，此时可以访问到 DOM 元素
  3. **updated**：【更新】组件数据更新后调用，DOM 也已更新
  4. **destroyed**：【卸载】组件销毁前调用，可以在这里执行清理工作
