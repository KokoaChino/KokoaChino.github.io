Mybatis

MyBatis 是一个流行的 Java 持久层框架，主要用于简化数据库操作

它提供了一个半自动化的对象关系映射解决方案，允许开发者通过 XML 或注解方式将 SQL 语句与 Java 对象进行映射

Foreach 操作

  • collection：指定要迭代的集合
  • item：当前迭代的元素
  • index：当前迭代元素的索引位置
  • open：在生成的SQL语句中，在开始生成列表元素之前添加的字符串
  • separator：在迭代元素之间插入的字符串
  • close：在生成的SQL语句中，在结束生成列表元素之后添加的字符串

二级缓存

  • flushInterval：用于控制缓存刷新时间，当到达指定时间时会自动清理所有缓存

  • readOnly：是否设置为只读

  • size：最大的缓存对象数量，当缓存达到上限时，会根据eviction配置的策略进行清理

eviction 策略 说明
LRU 最近最少使用 移除最长时间不被使用的对象
FIFO 先进先出 按对象进入缓存的顺序来移除它们
SOFT 软引用 基于垃圾回收器状态和软引用规则移除对象
WEAK 弱引用 更积极地基于垃圾收集器状态和弱引用规则移除对象
  • useCache：是否启用二级缓存
  • flushCache：在操作执行完成后，是否直接清除所有缓存
