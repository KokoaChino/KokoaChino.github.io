Maven 依赖字段 说明

artifactId 项目在当前组中的唯一名称
version 项目版本
type 依赖的类型
scope 依赖的作用域
optional 标记依赖是否可选
exclusions 用来排除传递性依赖

Maven 依赖的作用域

Maven 依赖的作用域 说明
compile 在编译、运行、测试时均有效【默认选项】
provided 在编译、测试时有效，但是在运行时无效
runtime 在运行、测试时有效，但是在编译代码时无效
test 只在测试时有效

Maven 常用命令

Maven 常用命令 说明
clean 清理整个目标文件夹
validate 验证项目的可用性
compile 将项目编译为 .class 文件
install 将当前项目安装到本地仓库
verify 按顺序执行每个默认生命周期阶段
test 一键测试所有位于 test 目录下的测试案例
package 直接对项目的代码进行打包，生成 jar 文件
