模式 描述 备注
'r' 只读 如果文件不存在，会抛出FileNotFoundError
'w' 只写 如果文件存在，会覆盖原文件；如果文件不存在，会创建新文件
'a' 追加 如果文件不存在，会创建新文件；如果文件存在，会将数据追加到文件末尾
'x' 独占创建 如果文件已存在，会抛出FileExistsError
'b' 二进制 用于读取/写入二进制数据，如图片、音频等。可以与其他模式结合使用
't' 文本 读取或写入的是字符串，而不是二进制数据。可以与其他模式结合使用
'r+' 读写 如果文件不存在，会抛出FileNotFoundError
'w+' 读写 如果文件存在，会覆盖文件；如果文件不存在，会创建新文件
'a+' 读写 如果文件存在，数据将被追加到文件末尾；如果文件不存在，会创建新文件
编码格式 描述
'utf-8' 常用的 Unicode 编码格式，支持世界上几乎所有语言字符
'utf-16' Unicode 编码格式，使用 2 或 4 字节表示字符
'utf-32' Unicode 编码格式，使用 4 字节表示字符
'ascii' 仅支持 7 位字符集，主要用于英语
'latin-1' 也称为 ISO-8859-1，主要用于西欧语言
'gbk' 主要用于简体中文，向后兼容 GB2312 编码
'gb18030' 中国大陆的标准编码，支持所有汉字字符
'big5' 繁体中文字符集，主要在台湾和香港使用
'cp1252' Windows 系统常用的西欧语言编码格式
'shift-jis' 日语的编码格式，广泛用于日本地区
'euc-jp' 另一种日语编码格式，主要用于日本的 UNIX 系统
'koi8-r' 俄罗斯的标准编码，主要用于俄语字符集
'mac-roman' 苹果电脑常用的字符编码，主要用于西欧语言

DataFrame

一个二维的表格数据结构，类似于 Excel 表格或 SQL 表格

它由多列数据组成，每一列可以是不同的数据类型，行和列都有索引
