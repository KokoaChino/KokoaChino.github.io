@RequestMapping

path / value：当前方法处理的请求路径

路径还支持使用通配符进行匹配：

  • ?：表示任意一个字符
  • *：表示任意 0 - n 个字符
  • **：表示当前目录或基于当前目录的多级目录

method：请求的方法类型

params：指定请求必须携带哪些请求参数（支持表达式）

header：指定请求头必须携带哪些请求参数（支持表达式）

@RequestBody

设置required = false，将属性设定为非必须

设置defaultValue，表示没有该属性时的一个默认值

Bean 的 Web 作用域

request

对于每次 HTTP 请求，使用 request 作用域定义的 Bean 都将产生一个新实例，请求结束后 Bean 也消失

session

对于每一个会话，使用 session 作用域定义的 Bean 都将产生一个新实例，会话过期后 Bean 也消失
