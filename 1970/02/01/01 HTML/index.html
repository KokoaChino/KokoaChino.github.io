HTML

HTML（超文本标记语言）是用于创建网页和 Web 应用程序的标准标记语言，它提供了一种结构化的方式来组织和展示文档内容

HTML 基本骨架

html 标签

定义 HTML 文档

标签限定了文档的起始点和结束点

head 标签

定义文档的头部

文档的头部描述了文档的各种属性和信息，包括文档的标题、在Web中的位置以及和其他文档的关系等

title 标签

定义文档的标题

meta 标签

定义文档的属性

link 标签

引入其他文件

body 标签

定义文档的主体

body元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等。）

它会直接在页面中显示出来，也就是用户可以直观看到的内容

标题

快捷键

生成 h1 ~ h6：h$*6

文本

文本标签 描述
<span> 没有特定的含义
<b> 粗体文本
<i> 斜体字
<del> 删除字
<em> 着重文字
<strong> 加重语气

段落

换行

水平线

属性：

  • color：颜色
  • width：宽度
  • size：高度
  • align：对齐方式（默认居中）

超文本链接

属性：

  • href：链接地址
  • download：下载文件的命名

列表

无序列表

type 属性：

  • disc：实心圆（默认选项）
  • circle：空心圆
  • square：小方块
  • none：不显示

列表是可以进行嵌套的

快捷键

生成 ul + li 的布局：ul>li*n（n 为需要的 li 数量）

有序列表

type 属性：

  • 1：表示列表项目用数字标号（1, 2, 3…）
  • a：表示列表项目用小写字母标号（a, b, c…）
  • A：表示列表项目用大写字母标号（A, B, C…）
  • i：表示列表项目用小写罗马数字标号（i, ii, iii…）
  • I：表示列表项目用大写罗马数字标号（I, II, III…）

列表是可以进行嵌套的

快捷键

生成 ol + li 的布局：ol>li*n（n 为需要的 li 数量）

表格

表格标签：

  • 表格：<table>
  • 表首行：<thead>
  • 文本行：<tbody>
  • 行：<tr>
  • 单元格（列）：<td>

表格属性：

  1. border：表格的边框
  2. width：表格的宽度
  3. height：表格的高度

快捷键

生成表格结构：table>tr*2>td{单元格}

单元格合并

属性：

  • 水平合并：colspan
  • 垂直合并：rowspan

表单

表单是由容器和控件组成的，一个表单一般应该包含用户填写信息的输入框，按钮等

这些输入框，按钮叫做控件，表单就是容器，它能够容纳各种各样的控件

属性：

  • action：服务器地址
  • name：名称
  • method：数据提交方式

表单元素

输入框

按钮

当用户单击确认按钮时，表单的内容会被传送到另一个文件

单选框

多选框

下拉列表

图片

属性：

  • src：路径
  • alt：替代文本
  • width：宽度
  • height：高度
  • title：鼠标悬停在图片上的给予提示

图片路径

绝对路径

电脑的盘符存储与访问的具体地址

相对路径

两者相对关系，两者在同⼀路径下可以直接访问

  1. 子级关系: /

  2. 父级关系: ../

  3. 同级关系: ./（可以省略）

网络路径

具体的⽹络地址

视频

添加视频嵌入代码即可

块级元素与行内元素

块级元素 行内元素
会在页面中独占一行（自上向下垂直排列） 不会独占页面中的一行，只占自身的大小
可以设置width，height属性 设置width，height属性无效
⼀般块级元素可以包含行内元素和其他块级元素 ⼀般行内元素包含行内元素，但不包含块级元素

常见块级元素：

div、form、h1 ~ h6、hr、p、table、ul 等

常见行内元素：

a、b、em、i、span、strong 等

行内块级元素（特点：不换行、能够识别宽高）

button、img、input 等

容器

HTML 中一个非常重要的块级元素，常用于将页面内容分组或包裹起来

通常用于布局和样式设计，以便于管理网页的结构

HTML5 新增标签

HTML5 新增标签：

  • <header></header>：头部

  • <nav></nav>：导航

  • <section></section>：定义文档中的节，如章节、页眉、页脚

  • <aside></aside>：侧边栏

  • <footer></footer>：脚部

  • <article></article>：代表一个独立的、完整的相关内容块，例如一篇完整的论坛帖子，一篇博客文章，一个用户评论等
